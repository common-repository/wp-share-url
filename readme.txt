== Description ==

* EN. Displays a link to the code to insert a link to your site at the end of each blog entry. There is nothing specifically to do this is not necessary - just activate the plugin, that's all. I will be happy with any ideas to improve functionality.
* RU. Выводит ссылку с кодом для вставки ссылки на ваш сайт в конце каждой записи блога. Ничего специально для этого делать не нужно, - просто активируйте плагин, вот и всё. Буду рад любым идеям по улучшению функционала.

Features:

1. Insert as direct link.
2. Insert as bbCode.
3. Insert as HTML.

== Installation ==

1. Unzip files.
2. Upload 'shareurl' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
